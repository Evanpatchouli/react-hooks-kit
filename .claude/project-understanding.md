# 项目理解

## 项目概述

**项目名称**: @evanpatchouli/react-hooks-kit
**当前版本**: 1.2.2
**项目类型**: React Hooks 工具库
**开发语言**: TypeScript
**许可证**: MIT

## 项目目标

这是一个开源的 React Hooks 工具库，旨在简化 React Hooks 的使用。提供约60个不同功能的 hooks，涵盖状态管理、解析器、监听器、副作用等多个方面。

## 核心特点

1. **轻量级**: 无外部依赖
2. **全面性**: 包含多种类型的 hooks（store、parser、listeners、effects等）
3. **TypeScript 支持**: 完整的类型定义和类型推断
4. **创新性**: 提供独特的 hooks 如 useWatch（部分监听）、useRipple（CSS Houdini 效果）等

## 版本管理规则

- z 版本号：单个测试通过时增加
- y 版本号：hook 通过所有测试或新增 hook 时增加
- x 版本号：整个模块通过测试时增加

## Hook 状态标记

- 无标记：完全稳定
- **new**: 已通过测试
- **cup**: 实验性
- **clock**: 计划中
